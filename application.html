Application migration table

Frameworks for consideration.

It's important to bear in mind that a move towards any of the frameworks below will require infrastructure changes to support it, as well as changes to the codebase to support modern modular Javascript. All of them work with NPM & Webpack. Without this initial legwork, we cannot move.

Sizing up exactly which of these frameworks is the right fit for us is a tricky endevour, and I'd really require more dedicated time to evaluate exactly which is the winner.
